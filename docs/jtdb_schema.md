# 日本外傷データバンク（JTDB）入力項目定義

> ソース: Trauma registryDataDefinition.pdf  
> AISバージョン: AIS 2005 Update 2008  
> スキーマバージョン: 1.0  
> 生成日: 2026-06-21

**必須度凡例**
- `○` コア項目（必須入力）
- `▲` オプション項目（入力なしでも可）
- `*` 自動計算項目
